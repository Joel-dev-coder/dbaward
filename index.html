📡 Live Guest Sync

Connect a free Firebase database so guests can see winners on their phones in real time as you reveal them. Create a free Firebase project →

Setup (2 minutes):
1. Go to console.firebase.google.com
2. Create a project → Add a Realtime Database → Start in test mode
3. ⚠️ In Rules tab, make sure read/write are both true:
{"rules":{".read":true,".write":true}}
4. Copy your Database URL (looks like https://your-project-default-rtdb.firebaseio.com)
5. Paste it below and click Connect
⚠️ Firebase not connected — guest sync works on same browser only (different tabs). For cross-device sync (phones), connect Firebase above.
